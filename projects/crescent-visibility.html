crescent-visibility

Browser visibility model explorer

The Python models were rewritten as client-side JavaScript so visitors can compare Ilyas, Yallop, Odeh, Shaukat, and SAAO-style labels without a server.

Open repo

This browser version uses an approximate astronomical estimator plus the repo's visibility decision thresholds. The Python/Skyfield workflow remains the source for high-precision ephemeris runs.
